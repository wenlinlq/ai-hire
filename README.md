# AI招新助手 - 智能校园招聘平台

## 项目简介

**AI招新助手** 是一款面向高校社团、实验室的智能招聘平台，利用人工智能技术，为招聘方和求职者提供全方位的智能化服务。平台整合了AI简历分析、AI模拟面试、AI智能问答、AI预面试和AI简历筛选等核心功能。

## 核心功能

### 1. AI简历分析与优化

基于大语言模型和OCR技术，自动解析和分析简历内容，提供专业的优化建议。

**功能特点：**

- **智能解析**：自动提取简历中的关键信息（支持PDF和图片格式）
- **技能匹配**：与岗位需求进行智能匹配，计算匹配度评分
- **优化建议**：根据岗位要求提供针对性的简历优化建议
- **缺陷检测**：识别简历中的常见问题

**技术实现：**

- 使用 `pdf-parse` 解析文本PDF
- 扫描型PDF和图片文件使用百度OCR提取文字
- 阿里云百炼大语言模型进行内容分析

### 2. AI模拟面试

提供真实的面试模拟体验，帮助求职者提升面试技巧。

**功能特点：**

- **智能出题**：根据岗位要求自动生成面试问题
- **实时评估**：AI分析回答内容，提供评分和反馈建议
- **面试报告**：生成详细的面试评估报告，包括优缺点分析

**技术实现：**

- 阿里云百炼大语言模型生成面试问题
- DeepSeek API进行回答评分
- 支持Markdown格式反馈建议渲染

### 3. AI智能问答

在线智能客服，解答招聘相关问题。

**功能特点：**

- **自然对话**：支持自然语言交互
- **知识问答**：解答招聘流程、岗位要求、面试技巧等问题
- **上下文记忆**：支持多轮对话，保留最近5条历史消息
- **流式响应**：SSE实时流式输出，打字机效果

**技术实现：**

- 阿里云百炼大语言模型
- SSE（Server-Sent Events）实现流式响应
- 前端维护对话上下文

### 4. AI预面试

在正式面试前进行AI预筛选，提高招聘效率。

**功能特点：**

- **自动筛选**：AI自动进行简历筛选和分析
- **标准化评估**：统一的评估标准，减少人为偏见
- **评分报告**：生成标准化的预面试评分报告

**技术实现：**

- 根据环境变量自动切换解析服务（开发/生产）
- 多维度评分模型

### 5. AI简历筛选

自动化简历筛选流程，快速找到合适候选人。

**功能特点：**

- **硬性条件筛选**：自动过滤不符合基本要求的简历
- **智能排序**：按匹配度对简历进行排序
- **批量处理**：支持批量上传和处理简历

**技术实现：**

- 规则引擎进行条件筛选
- MongoDB索引优化查询性能

## 技术栈

### 前端技术

- **框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式**: TailwindCSS 3
- **状态管理**: React Context + useState
- **路由**: React Router
- **图标**: Lucide React
- **HTTP客户端**: Axios

### 后端技术

- **框架**: Node.js + Express
- **数据库**: MongoDB
- **AI服务**: 阿里云百炼 / DeepSeek / 百度OCR
- **文档处理**: pdf-parse
- **认证**: JWT（单一Token机制）
- **跨域**: CORS

## 核心业务流程

### 学生求职流程

1. **注册登录** → 2. **完善简历** → 3. **AI简历分析** → 4. **浏览岗位** → 5. **投递申请** → 6. **AI预面试** → 7. **面试邀请** → 8. **正式面试**

### HR招聘流程

1. **创建岗位** → 2. **设置筛选规则** → 3. **接收投递** → 4. **AI简历筛选** → 5. **查看分析报告** → 6. **发送面试邀请** → 7. **安排面试** → 8. **录用决策**

## 安全保障

- **数据加密**: 用户数据传输全程加密（HTTPS）
- **访问控制**: 基于角色的权限管理（RBAC）
- **审计日志**: 完整的操作记录和审计追踪
- **JWT认证**: 无状态认证机制

## 部署方式

### 开发环境

```bash
# 安装依赖
cd client && npm install
cd ../server && npm install

# 启动后端服务
cd server && npm start

# 启动前端服务（新终端）
cd client && npm run dev
```

### 环境变量配置

后端 `.env` 文件需要配置：

```env
PORT=3000
MONGODB_URI=mongodb://localhost:27017/ai-hire
JWT_SECRET=your_jwt_secret
BAIDU_API_KEY=your_baidu_api_key
BAIDU_SECRET_KEY=your_baidu_secret_key
ALIYUN_API_KEY=your_aliyun_api_key
DEEPSEEK_API_KEY=your_deepseek_api_key
NODE_ENV=development
```

前端 `.env.production` 文件：

```env
VITE_API_URL=http://your-server-ip:3000
```
