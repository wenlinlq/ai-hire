# AI招新助手 - 智能校园招聘平台

## 项目简介

**AI招新助手** 是一款面向高校社团、实验室和企业的智能招聘平台，利用先进的人工智能技术，为招聘方和求职者提供全方位的智能化服务。平台整合了AI简历分析、AI模拟面试、AI智能问答、AI预面试和AI简历筛选等核心功能，致力于提升招聘效率和求职体验。

## 核心功能

### 1. AI简历分析与优化

基于深度学习技术，自动解析和分析简历内容，提供专业的优化建议。

**功能特点：**

- **智能解析**：自动提取简历中的关键信息
- **技能匹配**：与岗位需求进行智能匹配，计算匹配度评分
- **优化建议**：根据岗位要求提供针对性的简历优化建议
- **缺陷检测**：识别简历中的常见问题

**技术实现：**

- 使用自然语言处理（NLP）技术进行文本分析
- 深度学习模型进行技能实体识别
- 相似度算法进行技能匹配评分

### 2. AI模拟面试

提供真实的面试模拟体验，帮助求职者提升面试技巧。

**功能特点：**

- **智能出题**：根据岗位要求自动生成面试问题
- **实时评估**：AI实时分析回答内容，提供即时反馈
- **多轮对话**：支持多轮面试对话，模拟真实面试场景
- **面试报告**：生成详细的面试评估报告，包括优缺点分析

**技术实现：**

- 大语言模型进行问题生成和回答评估
- 语音识别技术支持语音回答
- 情感分析技术评估回答质量

### 3. AI智能问答

24/7在线的智能客服，解答招聘相关问题。

**功能特点：**

- **自然对话**：支持自然语言交互，无需固定指令
- **知识问答**：解答招聘流程、岗位要求、面试技巧等问题
- **个性化推荐**：根据用户情况提供个性化求职建议
- **多语言支持**：支持中英文问答

**技术实现：**

- 基于大语言模型的对话系统
- 知识库检索增强生成（RAG）技术
- 意图识别和实体抽取

### 4. AI预面试

在正式面试前进行AI预筛选，提高招聘效率。

**功能特点：**

- **自动筛选**：AI自动进行第一轮面试筛选
- **标准化评估**：统一的评估标准，减少人为偏见
- **时间节省**：HR只需关注通过预面试的候选人
- **评分报告**：生成标准化的预面试评分报告

**技术实现：**

- 自动化面试流程控制
- 多维度评分模型
- 决策树算法进行筛选判断

### 5. AI简历筛选

自动化简历筛选流程，快速找到合适候选人。

**功能特点：**

- **硬性条件筛选**：自动过滤不符合基本要求的简历
- **智能排序**：按匹配度对简历进行排序
- **批量处理**：支持批量上传和处理简历
- **可视化分析**：提供简历筛选数据可视化报告

**技术实现：**

- 规则引擎进行条件筛选
- 机器学习模型进行匹配度预测
- 大数据分析技术进行统计分析

## 技术栈

### 前端技术

- **框架**: React 18 + TypeScript
- **样式**: TailwindCSS 3
- **状态管理**: React Context + useState
- **路由**: React Router
- **图标**: Lucide React
- **HTTP客户端**: Axios

### 后端技术

- **框架**: Node.js + Express
- **数据库**: MongoDB
- **缓存**: Redis
- **AI服务**: 阿里云百炼 / 百度文心一言
- **文档处理**: 百度智能云文档服务
- **认证**: JWT

### AI技术

- **自然语言处理**: 大语言模型 API
- **文档解析**: OCR + NLP技术
- **语音识别**: ASR技术
- **机器学习**: 分类、回归、聚类算法

## 核心业务流程

### 学生求职流程

1. **注册登录** → 2. **完善简历** → 3. **AI简历分析** → 4. **浏览岗位** → 5. **投递申请** → 6. **AI预面试** → 7. **面试邀请** → 8. **正式面试**

### HR招聘流程

1. **创建岗位** → 2. **设置筛选规则** → 3. **接收投递** → 4. **AI简历筛选** → 5. **查看分析报告** → 6. **发送面试邀请** → 7. **安排面试** → 8. **录用决策**
